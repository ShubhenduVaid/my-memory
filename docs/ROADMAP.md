# Roadmap

This document outlines planned features and improvements for My Memory.

## Current Version (1.0.0)

- ✅ Spotlight-style search UI
- ✅ Semantic search with AI
- ✅ Apple Notes integration
- ✅ Obsidian integration
- ✅ Local files integration
- ✅ Notion integration
- ✅ Multiple LLM providers (Gemini, OpenRouter, Ollama)

## Short-term (v1.1)

- [ ] Improved search result ranking
- [ ] Keyboard shortcuts customization
- [ ] Dark/light theme toggle
- [ ] Search history

## Medium-term (v1.2)

## Long-term (v2.0)
