## [1.0.0] - 2026-01-18

### Added
- Initial release
- Spotlight-style search UI (`Cmd+Shift+Space`)
- Semantic search powered by Gemini AI
- AI-generated answers from your notes
- Apple Notes integration (macOS)
- Obsidian vault integration
- Local files integration (markdown, text, PDF)
- Notion integration
- Menu bar app with tray icon
- Auto-updates via GitHub Releases
- SQLite-based local caching
- Multiple LLM provider support (Gemini, OpenRouter, Ollama)
